# FENRIX

**FENRIX** is a modern high-performance UCI chess engine.

## Vision

FENRIX aims to become a strong, efficient, and cleanly designed open-source chess engine focused on deep calculation, smart evaluation, and long-term scalability.

## Features (Planned)

- UCI protocol support
- Alpha-Beta search
- Iterative deepening
- Transposition tables
- Advanced move ordering
- NNUE support (future)
